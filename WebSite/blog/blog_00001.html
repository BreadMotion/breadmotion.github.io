自己紹介

こんにちは!パン君です。

私はとあるゲーム会社で働きながら、インディーズや個人制作でも活動しているゲームエンジニアです。

社会人2年目ということもあり、インディーズやプライベートで制作した作品などを整理して
今後の活動に役立つような「まとめの場所」が欲しいと思いこのサイトを作りました。

本当はサーバーを借りたり、常駐用のミニPCを用意して1から環境構築して……という方法も考えましたが、
まずは 「早く出す」ことを優先 して、今回は GitHub Pages を使っています。

ページのクオリティは自分次第なので、今後もいろいろ調べながらブラッシュアップしていく予定です。
その過程も含めて、このサイトを通じて成長していけたらいいなと思っています。

やりたいこと

このサイトの大きな目標は下記のとおりです。

  • ポートフォリオを、将来的に「商売にも使えるレベル」のものに育てる
  • ポートフォリオを起点に、仕事の依頼やプロジェクトの相談が来る状態を目指す

技術ブログを作成した狙いとしては、

  • 作成したプロジェクトの詳しい機能紹介
  • 個人で作ったツールや仕組みの解説

といった内容をまとめておき、
ポートフォリオの補足資料 としても使えるようにしたい、という思いがあります。

ブログの記事自体も、長い目で見れば スキルと実績を示すポートフォリオの一部 になるので、
「これはお得だな」と感じて始めました。

このサイトの構成イメージ

現時点で想定している構成はこんな感じです。

ポートフォリオ
: 進捗報告や作品の掲載、関連するブログ記事の一部をまとめる場所

ブログ
: 技術メモ、開発中の作品の一部機能紹介、ハマりどころの整理など

プロジェクト一覧
: 参加・制作したプロジェクトの一覧と簡単な説明、必要に応じて告知など

サービス一覧
: 筆者が提供できること・相談可能な内容をまとめたページ
（どういう形でお手伝いできるかを整理）

スキル一覧
: 触れる技術や得意な領域を整理したスキルシート的なページ

コンタクト
: 活動場所や連絡先など、筆者へのアクセス情報をまとめたページ

サイトの不備について

  • サイトのシステム的な不備
    → 気づき次第、随時修正していきます。
  • 記載している URL を悪用した迷惑行為
    → 発覚次第、可能な範囲で対応・対策を行います。

最後に

最後まで読んでくださり、ありがとうございました！

まだ立ち上げたばかりのサイトですが、
今後も技術記事や制作物の紹介などを更新していくつもりです。

たまに覗いてもらえたら、とても嬉しいです。
これからもよろしくお願いします。
