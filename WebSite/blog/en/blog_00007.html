What can this flow do?

This flow visualizes the following series of tasks in a Windows environment so that they can be executed without mistakes.

  • Installation of rclone
  • Google Drive remote settings
  • How to get Drive Folder ID

You can eliminate "stuck points in building a Git LFS substitute environment in Windows" for yourself and your team.

Overview

This is how to share files so that they can be managed outside the limits of Git LFS in a Windows environment. The rough flow is as follows.

  • Introduction of WinGet
  • Introduction of RClone
  • RClone settings
